cursorproperty
cursor:String  [read-write]

Optional. Breaks the results into pages. A single page contains 20 lists. Provide a value of -1 to begin paging. Provide values as returned to in the response body's next_cursor and previous_cursor attributes to page back and forth in the list. Example: cursor=-1 Example: cursor=-1300794057949944903
